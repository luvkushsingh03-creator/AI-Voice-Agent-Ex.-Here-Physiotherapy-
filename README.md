# AI Voice Agent for Physiotherapy Centre

An AI-powered voice assistant designed to act as a **24/7 virtual receptionist for a physiotherapy clinic**.  
It can answer patient queries, provide clinic information, and assist with appointment-related questions.

# Features
- 24/7 automated call handling
- Answers physiotherapy-related queries
- Provides clinic information
- Handles appointment inquiries
- Natural conversational voice interaction

# Tech Stack
- Vapi AI
- Speech-to-Text
- Text-to-Speech
- Conversational AI
- Prompt Engineering

# Use Case
Many physiotherapy clinics miss calls outside working hours.  
This AI voice agent acts as a **virtual receptionist**, ensuring patients always receive assistance.
