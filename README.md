# AI Voice Agent for Physiotherapy Centre
This project demonstrates an AI-powered voice assistant designed to act as a **24/7 virtual receptionist for a physiotherapy clinic**. The system can handle patient queries, provide clinic information, and assist with appointment-related questions.

# Project Background
This AI Voice Agent was developed using **Vapi.ai** as part of an **AI Product Builder Workshop conducted in collaboration with The Economic Times**.

# Features
- 24/7 automated call handling
- Natural conversational AI interaction
- Handles patient queries
- Provides clinic information
- Simulates a virtual receptionist

# Scalability and Multi-Domain Applications
Although this prototype is designed for a **physiotherapy clinic**, the same AI voice agent architecture can be applied to many other domains such as:

- Healthcare clinics and hospitals
- Restaurants for reservations
- Customer support call centers
- Real estate inquiry handling
- Hotel booking assistance
- E-commerce customer service
- Educational institutions for student queries

This demonstrates how **AI voice agents can automate front-desk communication across multiple industries**, improving availability and reducing operational workload.
